DOCUMENTAÇÃO DA BIBLIOTECA PANDAS:
http://pandas.pydata.org/pandas-docs/stable/

Após baixar o zip da pasta, descompactar em um diretório conhecido.

Após instalar o Anaconda a inicialização do Jupyter é feita através 
do terminal (console do sistema operacional) utilizando o comando "jupyter notebook"

Após abrir o jupyter procure nele pela pasta onde foi descompactado o zip
e abra o arquivo Separador_Dados_Excel.ipynb

Execute todas as células do relatório com shift + enter, na última onde há um código comentado
retire o comentário e coloque os parâmetros que achar adequada para a separação.
